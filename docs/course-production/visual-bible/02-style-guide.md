# Visual Bible · Part 2 — Style Guide (LOCKED 2026-07-02, from Jarrad)

**Canonical reference images:** `/Users/jarradhenry/Sites/BMH apps/BMH Institute/docs/design/` (`image.png`, `image (1).png`, `image (2).png`, `image2`)
**Style name (working):** BMH Sticker System

## The look
Flat sticker-sheet illustration. Cornflower-blue field; objects in yellow / orange / cream / white with thick black hand-drawn outlines. Icon-first, never realistic. Tiny characters with dot eyes. Floating compositions, not scenes. Decorative doodles (sparkles, clouds, speech bubbles, grass tufts) as rhythm.

## Palette
- Background: cornflower blue (from reference images)
- Fills: golden yellow, orange, cream, white, black accents
- **6–8 colors max, flat fills only. No gradients, no texture, no lighting, no shadows, no perspective rendering.**

## The 15 rules (Style DNA — verbatim from Jarrad's spec)
1. **Every object is a sticker** — complete silhouette/outline/balance; croppable; the page is a collection of stickers, not a scene
2. **Icons, not illustrations** — design the icon that represents the object; never attempt realism
3. **Recognition beats accuracy** — minimum information for instant recognition; simplify past that
4. **One universal visual language** — mansion, mailbox, mug: identical construction rules; importance comes from placement, not detail
5. **Uniform complexity** — no hero object with 10× detail
6. **Imperfection is required** — uneven curves, wobble, relaxed proportions; never mathematically perfect
7. **Ignore materials** — wood/glass/steel render identically; only color differs
8. **Strong silhouettes** — recognizable as a black silhouette; avoid busy interiors
9. **Blue space is part of the design** — active negative space; don't fill every gap
10. **Decorative rhythm** — stars, sparkles, hearts, birds, motion marks, clouds, dots, grass, speech bubbles — placed for balance, never random
11. **Endless expandability** — every composition should accept 20 more icons without breaking
12. **Editorial before decorative** — every icon communicates; decoration only supports
13. **Animation friendly** — bounce, pop, rotate, wiggle, sway, scale, slide, float; nothing needing rigs
14. **Five-second test** — topic, mood, symbols instantly readable
15. **Sticker-sheet test** — cut every object out individually; each must work alone

## Characters in this system
Tiny scale · dot eyes · minimal facial features · cylindrical limbs · no anatomical detail. Identity = **silhouette + hair shape + skin tone + one outfit color + one signature prop** (see cast sheet v2).

## QA checklist (apply to every render)
Palette: no gradients/texture, ≤8 colors, flat fills. Lines: thick black, slight wobble, consistent weight, rounded corners. Objects: primitive geometry, minimal detail, strong silhouettes, sub-second readable, standalone sticker quality. Composition: floating layout, no continuous scene, even rhythm, generous negative space, doodles distributed. Characters: tiny, dot eyes, minimal features, cylindrical limbs. Rendering: no lighting/reflections/shadows/perspective. Motion: every object independently animatable.

## Final principle
The style is defined by its rules, not its subject matter — swap every icon for another industry's and it must still read as the same family.
