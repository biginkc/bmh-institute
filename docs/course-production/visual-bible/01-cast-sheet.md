# Visual Bible · Part 1 — Cast Sheet (v2 — BMH Sticker System)

**Rewritten 2026-07-02 for the locked sticker style (see 02-style-guide.md). v1 (photoreal) is superseded.**
In this system a character is NOT a face — it's a repeatable icon recipe: **silhouette + hair shape + skin tone + one outfit color + one signature prop.** Dot eyes, minimal features, cylindrical limbs, tiny scale. These recipes are LAW once approved.

## D. Rules that ride along
1. Skin tones render as flat fills from the palette's warm range — distinct per character, consistent forever
2. Signature prop appears in EVERY appearance of that character (it's half the identity at sticker scale)
3. Stressed-not-destitute translates here as: droopy posture, a single sweat-drop doodle, tilted speech bubble — never squalor iconography
4. No character ever gets more detail than the others (rule 5, uniform complexity)
5. Approved = frozen. No re-casting mid-course.
