# Quiz distractor checker rules

Review each distractor adversarially. Judge all of the following:

- Is it genuinely plausible to a learner?
- Is it definitively wrong according to the source back and course material?
- Is it neither a synonym nor a rewording nor a partial truth of the source back?
- Does its grammatical form, specificity, and length match the correct answer?
- Does it avoid unsupported facts?
- Does the complete item leave the source back unambiguously the single best answer?

Return one verdict per entry:

- `pass` when every check succeeds.
- `revise` for a fixable defect. Name the defect and identify the affected distractor.
- `needs_human_review` when the card requires escalation.

Never edit distractors. The checker reports findings only.
